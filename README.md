# Reset-ManagementState
Bulk-Reset management state on WYZE / DELL / HP P25 ZeroClients

Useful when having to bulk-reset clients while updating from an very old MC version to current, when the MC's SSL Cert changed, or Apocalypse in general.
May or may not work on other TERA2 based Firmwares. Only tested and developed for P25 Platform

# Usage
Double Click to run script.
